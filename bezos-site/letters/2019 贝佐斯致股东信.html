股东信

2019 贝佐斯致股东信

致股东：

这次新冠疫情让我们深刻认识到，亚马逊对客户来说已经变得多么重要。我们希望你知道，我们严肃对待这份责任，并为我们的团队在这个艰难时期帮助客户所做的工作感到自豪。

疫情应对

亚马逊人正在日夜不停地工作，将必需品直接送到需要它们的人们家门口。我们看到的对基本生活用品的需求非常巨大，而且持续至今。但与可预测的假日高峰不同，这次激增几乎毫无预警，给供应商和配送网络带来了巨大挑战。我们迅速优先保障家庭必需品、医疗用品和其他关键产品的备货和配送。

我们的 Whole Foods 商店一直保持营业，为客户提供新鲜食品和其他重要商品。我们采取措施帮助最脆弱的群体，在 Whole Foods 每天的第一个小时专门留给老年人购物。我们临时关闭了 Amazon Books、Amazon 4-star 和 Amazon Pop Up 店铺，因为它们不销售必需品，并为这些关闭门店的员工提供了在亚马逊其他部门继续工作的机会。

在提供这些基本服务的同时，我们把员工和合同工的安全放在首位——我们对他们的英勇工作深怀感激，并致力于他们的健康和福祉。在医学专家和卫生部门的密切指导下，我们在运营网络和 Whole Foods 门店进行了150多项重大流程调整，以帮助团队保持健康，并每天审查这些措施。我们在全球各站点分发口罩、实施体温检测。我们定期消毒门把手、扶手、储物柜、电梯按钮和触摸屏。

我们还推出了广泛的社交距离措施。取消了换班时的站会，改为在布告栏分享信息，错开休息时间，在休息室拉开座位间距。

下一步可能是对所有亚马逊人进行常规检测，包括无症状者。如果每个人都能定期接受检测，将极大地改变我们对抗这种病毒的方式。检测阳性者可以被隔离和治疗，检测阴性者可以满怀信心地重返经济活动。

我们已经开始建设增量检测能力。一群亚马逊人——从研究科学家、项目经理到采购专家和软件工程师——从他们的日常工作转到了一个专门团队来推进这项工作。我们已经开始组装设备来建造第一个实验室，并希望很快开始对少量一线员工进行检测。我们不确定能在相关时间框架内走多远，但我们认为值得尝试，并随时准备分享我们学到的一切。

在探索长期解决方案的同时，我们也致力于当下支持员工。我们将美国的最低时薪临时增加了2美元，加拿大每小时增加2加元，英国每小时增加2英镑，许多欧洲国家每小时增加2欧元。加班费翻倍，最低每小时34美元。这些加薪仅到4月底就将花费超过5亿美元。虽然我们知道这很昂贵，但我们相信这是在当前情况下应该做的正确之事。我们还设立了亚马逊救济基金，初始资金2,500万美元，用于支持我们的独立配送服务合作伙伴及其司机、Amazon Flex 参与者和面临经济困难的临时工。

3月，我们在物流和配送网络新增了100,000个岗位。在成功填补这些职位后，我们又宣布创造75,000个新岗位以响应客户需求。我们知道世界上很多人因为失业或停薪留职而遭受经济困难。我们很高兴在一切恢复正常之前欢迎他们加入我们的团队。

亚马逊也在积极保护客户免受不法分子利用危机的侵害。我们因新冠相关价格欺诈而下架了超过50万个产品，并在全球暂停了6,000多个违反公平定价政策的卖家账户。

AWS 也在这次危机中发挥着重要作用。组织能够获得可扩展、可靠且高度安全的计算能力——无论是用于重要的医疗保健工作、帮助学生继续学习，还是让前所未有数量的员工在家高效工作——在这种情况下至关重要。医院网络、制药公司和研究实验室正在使用 AWS 来护理患者、探索治疗方案并以多种方式减轻新冠的影响。全球的学术机构正在从线下课堂转向虚拟课堂，依靠 AWS 确保学习的连续性。各国政府正利用 AWS 作为安全平台来构建新能力，以结束这场大流行。

我们正在与世界卫生组织合作，提供先进的云技术和技术专长来追踪病毒、了解疫情并更好地控制传播。我们还启动了 AWS 诊断开发计划，承诺2,000万美元来加速客户研发更准确的新冠诊断解决方案。

Alexa 健康团队开发了一个功能，让美国客户在家评估自己的新冠风险等级。客户可以问："Alexa，如果我觉得自己得了新冠该怎么办？"Alexa 会就症状和可能的接触史进行一系列提问，然后根据回答提供基于 CDC 来源的指导。

超越新冠

尽管这是极其艰难的时期，但它们也是一个重要提醒：我们作为一家公司所做的事情能对人们的生活产生巨大影响。客户依赖我们，而我们有幸能够提供帮助。凭借我们的规模和快速创新的能力，亚马逊能产生积极影响，成为推动进步的力量。

气候承诺

去年，我们与联合国前气候变化负责人 Christiana Figueres 及 Global Optimism 共同创立了 气候承诺（The Climate Pledge），并成为第一个签署方。该承诺要求亚马逊比《巴黎协定》的目标提前10年——在2040年实现净零碳排放。亚马逊要实现这个目标面临重大挑战，因为我们不只是传输信息——我们有庞大的物理基础设施，每年在全球配送超过100亿件商品。我们相信，如果亚马逊能提前十年做到，任何公司都能做到——我们希望与所有公司合作让它成为现实。

为此，我们正在招募其他公司签署 气候承诺。签署方同意定期测量和报告温室气体排放，实施与《巴黎协定》一致的脱碳战略，并在2040年实现年度净零碳排放。

我们计划通过购买 Rivian（一家密歇根州的电动汽车制造商）的100,000辆电动配送货车来部分实现这一承诺。亚马逊的目标是最早在2022年让10,000辆 Rivian 新电动货车上路，到2030年全部100,000辆上路。这对环境有利，但意义更为深远——这类投资向市场发出信号：企业应该开始发明和开发大型全球公司向低碳经济转型所需的新技术。

我们还承诺到2024年实现80%可再生能源、到2030年实现100%可再生能源。（团队实际上在推动2025年就达到100%，并且有一个有挑战性但可信的计划。）在全球范围内，亚马逊有86个太阳能和风能项目，发电容量超过2,300兆瓦，每年产出超过630万兆瓦时的能源——足以为超过580,000个美国家庭供电。

十多年前我们创建了"无忧包装"项目。自2008年以来，该项目节省了超过810,000吨包装材料，减少了14亿个快递箱的使用。

我们的可持续发展科学家花了三年多时间开发测量碳足迹的模型、工具和指标。他们的详细分析发现，网购产生的碳排放始终低于开车去商店购物——因为一辆配送车的单次行程平均可以替代大约100次私家车往返。

AWS 也天然比传统自建数据中心更高效。451 Research 的研究发现，AWS 的基础设施能效是被调查的美国企业数据中心中位数的3.6倍。加上可再生能源的使用，AWS 以比传统数据中心低88%的碳足迹完成同样的工作。

利用规模做善事

过去十年，没有任何公司比亚马逊创造了更多的就业机会。亚马逊在全球直接雇用840,000名员工，在美国直接和间接支持200万个就业岗位。在全球，我们支持近400万个就业岗位。我们特别自豪的是，其中很多是入门级岗位，为人们提供了参与劳动力市场的第一次机会。

亚马逊的工作岗位配备行业领先的每小时15美元最低工资和全面的福利。超过4,000万美国人——其中很多人拿着联邦最低工资每小时7.25美元——的收入低于亚马逊最低薪员工。我们希望其他大型雇主加入我们，提高他们自己的最低工资，我们也继续游说设立15美元的联邦最低工资。

我们还在花7亿美元为超过100,000名亚马逊人提供医疗保健、云计算、机器学习等高需求领域的培训项目。自2012年以来，我们的 Career Choice 项目已为超过25,000名亚马逊人提供了在高需求职业方面的培训。

Amazon Future Engineer 项目旨在为低收入和弱势年轻人提供计算机科学教育和培训。我们有一个宏大的目标：每年帮助数十万学生学习计算机科学和编程。

就目前而言，我个人的时间和思考继续集中在新冠疫情以及亚马逊在其中能如何帮助。我对亚马逊同事们在这个过程中展现的坚韧和创造力深表感激。你们可以相信我们所有人都会超越眼前的危机，寻找洞见和教训，并将其应用到未来。

想一想苏斯博士（Theodor Seuss Geisel）的这句话：

"当坏事发生时，你有三个选择：让它定义你，让它摧毁你，或者让它使你变得更强大。"

我非常乐观地相信人类文明会选择哪一个。

即使在这样的境况下，现在仍然是 Day 1。一如既往，我附上了我们1997年最初的那封信。
