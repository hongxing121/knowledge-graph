人物

纳瓦尔

基���信息

  • 全名：Naval Ravikant（纳瓦尔·拉维坎特）
  • 角色：创业者、天使投资人、思想家、播客主持人
  • 背景：印度裔美国人，在纽约皇后区长大的单亲家庭孩子
  • 在素材中出现：17 篇（全部素材的主角）

人物画像

成长背景

纳瓦尔出生于印度，在纽约皇后区长大。单亲家庭，妈妈把图书馆当托儿所。他在那里读完了一切——杂志、地图、每一本书。他的阅读进化路径是：漫画 → 推理 → 奇幻 → 科幻 → 科学 → 数学 → 哲学。

"记住我一开始是印度的一个穷孩子，对吧？如果我能做到，任何人都能。"
——08 拯救自己

他部分在印度长大，记得印度祖父母家的场景——趴在地板上翻看祖父所有的《读者文摘》（参见 06 建立判断力）。印度大家庭的环境让他理解了人际关系的两面性：有人关心你让你不孤单，但也没有隐私和自由。

创业史

纳瓦尔在 1996 年左右在 @Home Network 工作时，到处告诉身边的人"我要创业"，最终被同事的催问逼上了创业之路（参见 05 专注、运气与耐心）。

他的第一家公司是 Epinions（消费者评价网站，约 1999 年），有过惨痛教训——赚到的财富"基本上被商业伙伴骗走了"。后来他起诉了对方，结果对他有利，但过程中的愤怒和焦虑让他深刻反思。

之后他和 Babak Nivi 共同创建了 Venture Hacks 博客，再演化为 AngelList——他最重要的商业项目。他还是 Twitter 和 Uber 的早期投资人（Twitter 约 400 倍回报，Uber 约 4000 倍回报）。

"我赚到的第一笔小财富，立刻就在股市里亏掉了。第二笔本该赚到的小财富，基本上被我的商业伙伴给骗走了。只��第三次才算成功。"
——05 专注、运气与耐心

思想体系

纳瓦尔的思想可以分为两大支柱：

财富哲学（Part I）：核心公式是产品化自己 = 专属知识 + 责任感 + 杠杆。他在 2018 年的推文风暴"How to Get Rich (without getting lucky)"中将这套体系浓缩为 38 条推文，后展开为 nav.al 播客系列。核心主张包括：出租时间永远不会致富；必须拥有所有权；三种杠杆中，代码与媒体是新型的无需许可杠杆；判断力在杠杆时代是最被低估的能力。

幸福哲学（Part II）：核心主张是幸福是一种技能——可以训练，不是天赋。幸福 = 内心平静 = 欲望的缺席。欲望是痛苦的契约。他的哲学深受佛教影响但以科学为基础，自称"理性佛教"——接受进化论和可证伪性，拒绝前世和不可验证的超自然声称。

"人生三大事：财富、健康、幸福。我们按这个顺序追求，但重要性恰好相反。"
——07 学习幸福

日常习惯

冥想：纳瓦尔的核心日常实践。他偏好"无选择觉知"（choiceless awareness）和每天一小时静坐。他已坚持超过两年半，称"当我一天中最好的一小时是独处时，这个世界对我就没什么可给的了"（参见 2020-10 Tim Ferriss 473 - 幸福的艺术）。

阅读：他的"初恋"和超能力。每天读一到两小时。同时在 Kindle 上打开 50-70 本书，不按顺序读，不强迫读完。宁可反复读 100 本经典也不泛读所有书。

"我大概每天读一到两小时。这大概能排进前 0.00001%。我觉得光凭这一点就能解释我生活中的所有物质成功。"
——2017-02 Knowledge Project 18 - 清醒思考

锻炼：每天早上锻炼是第一优先级，高于幸福、家庭和工作。"世界在燃烧也得等他做完。"

冷水淋浴：受 Wim Hof 启发，用冷水淋浴来训练接受力。核心教训："我们大部分的痛苦来自逃避。"（参见 08 拯救自己）

核心价值观

  1. 诚实 — "言行不一会创造多线程的心智负担，把你拉出当下"
  2. 长期主义 — "所有好处来自复利效应"，只和长期玩家打长期博弈
  3. 平等关系 — "不要高人一等，也不要低人一等"
  4. 远离愤怒 — "愤怒是你手里握着的热煤炭，等着扔向别人"
  5. 选择的自由 — 从年轻时的"自由去做"进化到现在的"免于……的自由"

"给年轻自己的建议：'做完完全全的你自己。'"
——08 拯救自己
